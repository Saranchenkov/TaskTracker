# TaskTracker
### How to run server:
####    1) create MySQL database with name 'task_tracker' and initialize (sql/schema.sql)
####   2) open /src/main/resources/application.yml and change necessary properties:
    
      mail.username
      mail.password
      
      datasource.username
      datasource.password
      
      frontend-server-url
####    3) connect to your database
####    4) run the server
